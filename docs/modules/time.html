fromRFC3339PartialTime

  • fromRFC3339PartialTime(str: string): CivilTime
    • Defined in time.ts:105

    Attempts to create a CivilTime object from a string using the RFC 3339 partial-time format. (e.g. 23:15:23.250) Since partial-time allows for leap seconds and CivilTime explicitly does not, this will interpret second 60 as the first second of the following minute, overflowing back to 00:00:00 if necessary.

    since

    0.1.0

    • str: string

    Returns CivilTime
